# Database

* beekeeper studio - [github repo](https://github.com/beekeeper-studio/beekeeper-studio) - SQL client
* dice - [github repo](https://github.com/DiceDB/dice) - in-memory database #go !archived
* dolt - [github repo](https://github.com/dolthub/dolt) - git for data
* electric - [github repo](https://github.com/electric-sql/electric) - a Postgres sync engine
* gephi - [github repo](https://github.com/gephi/gephi) - platform for visualizing and manipulating large graphs
* immudb - [github repo](https://github.com/codenotary/immudb) - built-in cryptographic proof and verification
* orbitdb - [github repo](https://github.com/orbitdb/orbitdb) - serverless, distributed, peer-to-peer database
* pgroll - [github repo](https://github.com/xataio/pgroll) - PostgreSQL zero-downtime migration
* planter - [github repo](https://github.com/achiku/planter) - PlantUML ER diagram from PostgreSQL tables
* questdb - [github repo](https://github.com/questdb/questdb) - time-series database, alternative to influxdb
* scylladb - [github repo](https://github.com/scylladb/scylladb) - real-time big data database, API-compatible with Apache Cassandra and Amazon DynamoDB
* sqlc - [github repo](https://github.com/sqlc-dev/sqlc)
* ydb - [github repo](https://github.com/ydb-platform/ydb) - distributed SQL database
