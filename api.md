# API

* bruno - [github repo](https://github.com/usebruno/bruno) - alternative to Postman
* django ninja - [github repo](https://github.com/vitalik/django-ninja)
* fastapi - [github repo](https://github.com/fastapi/fastapi) - web framework for building API
* postgrest - [github repo](https://github.com/PostgREST/postgrest) - RESTful API from any existing PostgreSQL database
* schemathesis - [github repo](https://github.com/schemathesis/schemathesis) - testing automation from openAPI schema
* trailbase - [github repo](https://github.com/trailbaseio/trailbase) - alternative to firebase
